Description

Unitto is a paid unit converter app for Android devices.

Goals

Make an easy to use yet powerful and customizable app that doesn't reinvent the wheel.

Design guidelines

Unitto's design is based on Google's Material Design 3 guidelines. As a lot of people are already familiar with language translators, such apps were used as design references: buttons placement, units selection screen, main app logic in general.

Key features

1. Designed different

Unitto is designed to keep it's users as efficient as possible, allowing them to convert units and resume their activity as quick as possible.

2. More than 150 units

Even though this number will grow, finding and choosing needed units stays simple and quick thanks to group filters, favorites filter and search functionality.

3. SI Standard*

For example, kilobyte is equal to 1000 bytes, not 1024. Many competitors fail to understand this user need.

4. Personalized experience

Unitto provides a wide range of different settings, which allows the application to adapt to each user individually.

App name

App is called "Unitto" after one of the variables in source code — "unitTo" — which comes from "unit to converter to"

Version codenames

Color names are used for codenames. They go in alphabetical order. Each update is unique and always makes the app better, bringing new features, improving UX and increasing app's stability.

  • Room

  • Material Design 3 Components
